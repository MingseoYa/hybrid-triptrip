# ✈️ 트립트립
혼자 여행하는 사람들을 위한 장소 공유 커뮤니티 앱

### 개발환경
- 개발 기간: 2024.11 ~ 2024.12 (약 3주)
- 

### 주요기능
- 로그인/회원가입
- 솔플레이스 등록/수정
- 솔플레이스 리스트 무한 스크롤
- 솔플레이스 등록 시 푸시알림
- 위치/알림 권한 요청
